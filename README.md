# Arknights Gacha Simulator
Gacha simulator for Arknights written in JavaScript for NodeJS.

Wrote this for fun. I may improve this when I have free time.

## Requirements
1. NodeJS

## How to use?
1. Download index.js and package-lock.json
2. Run npm install
3. Run node src/app
